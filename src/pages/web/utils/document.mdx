Document utils are a set of utility functions that are used to work with the **document** object in the browser environment. The **document** object represents the web page loaded in the current browser window or tab, and provides a way to interact with the HTML elements on the page.

## Create Element

The **createDocumentElement** function is used to create new elements in the **document** object in the browser environment. The function is a generic function that takes a **tagName** parameter and an optional **options** parameter.
