The **@aracna/react** package has dependencies on two other packages: **@aracna/core** and **@aracna/web**. These packages are required for the **@aracna/react** package to function properly.

## NPM

<CodeWindow client:load language='bash'>
  npm install --save @aracna/react
</CodeWindow>

## Yarn

<CodeWindow client:load language='bash'>
  yarn add @aracna/react
</CodeWindow>
