- **Events**: A generic type parameter that extends the **ElementComponentEvents** interface and specifies the event listeners of the DOM element.
- **Props**: A generic type parameter that extends the **ElementComponentProps** interface and specifies the props of the React component.
- **Key**: A generic type parameter that extends the **keyof** **HTMLElementTagNameMap** interface and specifies the tag name of the DOM element.
