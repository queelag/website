This hook performs a re-render when any of the observerd properties used inside the render function change.

### Usage

```tsx live
function App() {
  const store = useRef(observe({ number: 0 }))

  const onClick = () => {
    store.current.number++
  }

  return useObserver(() => (
    <button onClick={onClick}>{store.current.number}</button>
  ))
}
```
