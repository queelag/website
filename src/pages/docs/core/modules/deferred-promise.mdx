title: DeferredPromise
---

This class extends a Promise and makes it deferrable.

### Usage

```ts
import { DeferredPromise } from '@queelag/core'

const promise: DeferredPromise<number> = new DeferredPromise()

async function main(): Promise<void> {
  // will log 1
  console.log(await promise.instance)
}

main()

// will resolve the promise after 1s
setTimeout(() => promise.resolve(1), 1000)
```
