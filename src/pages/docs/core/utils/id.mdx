This class contains utilities for identifiers.

### Generate Prefixed

```ts
import { IDUtils } from '@queelag/core'

// will log john_$random
console.log(IDUtils.prefixed('john'))
```

### Generate Unique

```ts
import { IDUtils } from '@queelag/core'

const ids: string[] = [IDUtils.unique(), IDUtils.unique()]

// will log a $random different than the ones contained inside ids
console.log(IDUtils.unique(ids))
```
