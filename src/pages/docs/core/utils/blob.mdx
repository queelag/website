This document describes various Blob utility functions.

### Serialize

```ts
import { AracnaBlob, serializeBlob } from '@aracna/core'

const aracna_blob = new AracnaBlob(new Blob([]))
const blob = serializeBlob(aracna_blob)

// will log the Blob instance that was inside aracna_blob
console.log(blob)
```

### Deserialize

```ts
import { AracnaBlob, deserializeBlob } from '@aracna/core'

const blob = new Blob([])
const aracna_blob = deserializeBlob(blob)

// will log the AracnaBlob instance with the blob inside of it
console.log(aracna_blob)
```

### Deserialize With Resolution

```ts
import { AracnaBlob, deserializeBlob } from '@aracna/core'

const blob = new Blob([])
const aracna_blob = deserializeBlob(blob, {
  resolveArrayBuffer: true,
  resolveText: true
})

// will log the AracnaBlob instance with the blob inside of it
console.log(aracna_blob)

// will log the resolved ArrayBuffer
console.log(aracna_blob.arrayBuffer)

// will log the resolved text
console.log(aracna_blob.text)
```
