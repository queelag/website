This class is a dedicated Error instance for Fetch errors.

### Usage

```ts
import { Fetch, FetchError } from '@aracna/core'

const response = await Fetch.get('error')
if (response instanceof FetchError) console.log(response)

// will log response since it will be instance of FetchError
```
