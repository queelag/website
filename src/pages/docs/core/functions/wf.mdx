This function waits for the target function to return a truthy value or an error, "wf" stands for wait for.

### Usage

```ts
import { sleep, wf } from '@aracna/core'

let a = false

function check() {
  return a
}

sleep(1000).then(() => {
  a = true
})

// will stop waiting after 1s since a becomes truthy
console.log(await wf(() => check()))
```
