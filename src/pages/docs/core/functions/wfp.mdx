This function waits for the target async function to return a truthy value or an error, "wfp" stands for wait for promise.

### Usage

```ts
import { sleep, wfp } from '@aracna/core'

let a = false

async function check() {
  return a
}

sleep(1000).then(() => {
  a = true
})

// will stop waiting after 1s since a becomes truthy
console.log(await wfp(() => check()))
```
