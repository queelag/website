This function makes a function error safe, "mtc" stands for make try catch.

### Usage

```ts
import { mtc } from '@aracna/core'

function will_throw() {
  throw new Error()
}

const will_throw_safely = mtc(() => will_throw())

// will not throw and just return the error safely
console.log(will_throw_safely())
```
