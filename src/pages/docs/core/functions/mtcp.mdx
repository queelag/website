This function makes an async function error safe, "mtcp" stands for make try catch promise.

### Usage

```ts
import { mtc } from '@aracna/core'

async function will_throw() {
  throw new Error()
}

const will_throw_safely = mtcp(() => will_throw())

// will not throw and just return the error safely
console.log(await will_throw_safely())
```
