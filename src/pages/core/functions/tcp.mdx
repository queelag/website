This function makes try catching promises less verbose, "tcp" stands for try catch promise.

### Usage

```ts
import { tcp } from '@aracna/core'

// will log 0
console.log(tcp(async () => 0))

// will log Error
console.log(
  tcp(async () => {
    throw new Error()
  })
)
```
