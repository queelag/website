The **ma** function is a utility function that stands for "make async". It takes a synchronous function **fn** as its argument and returns a new function that is asynchronous and returns a **Promise**.

The **ma** function is a simple but powerful tool for making synchronous functions asynchronous, and for composing asynchronous functions together in an asynchronous programming style.
