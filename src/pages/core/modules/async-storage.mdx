The **AsyncStorage** class is an abstraction for any storage API, such as **localStorage** and **sessionStorage**. It provides a simple and flexible interface for working with a storage, allowing for getting, setting, removing, and copying items, as well as updating specific keys in an item.

## Set

The **set** method sets an item in the storage by its key, and optionally updates specific keys in the item. The method takes two parameters: **key** and **item**. The **key** parameter is a string that represents the key of the item to be set in the storage. The **item** parameter is an object that represents the item to be set in the storage.

## Remove

The **remove** method removes an item from the storage by its key, and optionally removes specific keys from the item. The method takes two parameters: **key** and **keys**. The **key** parameter is a string that represents the key of the item to be removed from the storage. The **keys** parameter is an optional array of strings that represent the keys to be removed from the item.

If the **keys** parameter is not provided, the entire item is removed from the storage. If the **keys** parameter is provided, only the specified keys in the item are removed, and the updated item is set back in the storage.

## Copy

The **copy** method copies an item from the storage to a target object, and optionally copies specific keys from the item. The method takes three parameters: **key**, **target**, and **keys**. The **key** parameter is a string that represents the key of the item to be copied from the storage. The **target** parameter is an object that represents the target object to which the item is to be copied. The **keys** parameter is an optional array of strings that represent the keys to be copied from the item.

If the **keys** parameter is not provided, all keys in the item are copied to the target object. If **the** keys parameter is provided, only the specified keys in the item are copied to the target object.

## Has

The **has** method checks if an item with the specified key exists in the storage, and optionally checks if specific keys exist in the item. The method takes two parameters: **key** and **keys**. The **key** parameter is a string that represents the key of the item to be checked in the storage. The **keys** parameter is an optional array of strings that represent the keys to be checked in the item.

If the **keys** parameter is not provided, the method checks if an item with the specified key exists in the storage. If the **keys** parameter is provided, the method checks if the specified keys exist in the item.
