The **Base32** class provides static methods for encoding and decoding data using the Base32 encoding scheme. The Base32 encoding scheme represents binary data as a sequence of 32 symbols (A-Z and 2-7).

The **Base32** class is a simple and useful utility class that provides a convenient way to encode and decode binary data using the Base32 encoding scheme.
