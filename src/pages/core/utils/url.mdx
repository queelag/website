URL utilities are a set of functions that provide useful functionality for working with URLs (Uniform Resource Locators) in ECMAScript. URLs are used to identify resources on the internet, such as web pages, images, and other files.

## Concatenate URL Chunks

The **concatURL** function is a utility function that takes one or more URL segments as input and concatenates them into a single URL string. The function removes any extra slashes between segments and trims whitespace.

## Append Search Parameters

The **appendSearchParamsToURL** function is a utility function that takes a URL string and a set of search parameters as input and appends the parameters to the URL as a query string. The function handles cases where the URL already contains a query string and removes any duplicate or unnecessary characters.

## Remove Search Parameters

The **removeSearchParamsFromURL** function is a utility function that takes a URL string as input and removes any query string parameters from the URL. The function returns the modified URL string.
