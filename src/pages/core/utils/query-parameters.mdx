Query parameters utils are a set of utility functions that are designed to help with working with query parameters in URLs in ECMAScript.

## Serialize

The **serializeQueryParameters** function is a utility function that takes an object with keys and values as input and returns a string of query parameters. The function iterates over the keys of the input object and creates an array of strings in the format **key=value**. It then joins the array with the & character to create the final string of query parameters.

## Deserialize

The **deserializeQueryParameters** function is a utility function that takes a string of query parameters as input and returns an object with the same keys and values. The function uses the **URLSearchParams** API to parse the input string into a **URLSearchParams** object, and then iterates over the entries of the object to create a new object with the same keys and values.
