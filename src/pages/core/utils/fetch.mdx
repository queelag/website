Fetch utils are a set of utility functions that are designed to make working with the Fetch API in ECMAScript easier and more efficient. The Fetch API is a modern interface for fetching resources over the network, and is commonly used in web applications to make HTTP requests to servers.

## Delete Request Init Header

The **deleteFetchRequestInitHeader** function is used to delete a specific header from a **FetchRequestInit** object. The function takes a **FetchRequestInit** object and a string representing the name of the header to delete as input, and returns a new **FetchRequestInit** object with the specified header removed.
