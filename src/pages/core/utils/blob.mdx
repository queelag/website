Blob utils are a set of utility functions that are designed to make working with **Blob** objects in ECMAScript easier and more efficient. **Blob** objects are used to represent binary data in web applications, such as images, audio files, and other types of media.

## Serialize

The **serializeBlob** function used to serialize an **AracnaBlob** object into a new **Blob** object. The function takes an **AracnaBlob** object and an optional **EndingType** value as input, and returns a new **Blob** object. The **EndingType** value is used to specify the line endings of the serialized **Blob**.

## Deserialize

The **deserializeBlob** function is used to deserialize a **Blob** object into an **AracnaBlob** object. The function takes a **Blob** object and an optional **DeserializeBlobOptions** object as input, and returns a new **AracnaBlob** object. The **DeserializeBlobOptions** object is used to specify whether the **ArrayBuffer** and/or text content of the **Blob** should be resolved during deserialization.
