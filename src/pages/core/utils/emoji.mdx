Emoji utils are a set of utility functions that are designed to make working with emojis in ECMAScript easier and more efficient. Emojis are graphical symbols that are used to represent emotions, objects, and other concepts in digital communication.

## Get Flag From Country Code

The **getEmojiFromCountryCode** function is used to generate a flag emoji from a two-letter country code. The function takes a string representing a two-letter country code as input, and returns a string representing the corresponding flag emoji.
