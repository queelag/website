This document describes various [Document](https://developer.mozilla.org/en-US/docs/Web/API/Document) utility functions.

### Create Element

```ts
import { createDocumentElement } from '@aracna/core'

// will log the div element if document is defined, otherwise it will return an empty object
console.log(createDocumentElement('div'))
```
