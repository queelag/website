The `Cookie` class is an abstraction to implement any cookie API in a uniform way.

## Properties

- **name (string):** the name of the cookie instance.
- **separator (string):** The separator used to separate the cookie name from the cookie item key. It defaults to `DEFAULT_COOKIE_SEPARATOR`.

## Set an item

The `set` method in the `Cookie` class sets an item in the cookies. Optionally, you can specify the keys of the item that you want to set. If you don't specify any key, the whole item will be set.
