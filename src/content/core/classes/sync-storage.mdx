The `SyncStorage` class is an abstraction to implement any synchronous storage API in an uniform way.

## Properties

- **name (string):** The name of the storage instance.

## Set an item

The `set` method in the `SyncStorage` class is an synchronous method that sets an item in the storage. Optionally, you can specify the keys of the item that you want to set. If you don't specify any key, the whole item will be set.
