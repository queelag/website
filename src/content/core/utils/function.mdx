Function utils are a set of utility functions that are used to work with functions in a programming language. These utility functions can be used to manipulate function arguments and return values, check if a value is an instance of a function, or perform other operations related to functions.

## Check Instance

The **isInstanceOf** function is a utility function for checking if a value is an instance of a given function. The function takes two parameters: a value to check and a function to compare against. The function returns a boolean value indicating whether the value is an instance of the function.
