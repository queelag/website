FormData utils are a set of utility functions that are designed to make working with **FormData** objects in ECMAScript easier and more efficient. **FormData** is a built-in ECMAScript object that provides a way to construct and send data in the format of a **multipart/form-data** request. This format is commonly used to submit forms that contain files, such as images or documents, to a server.

## Serialize

The **serializeFormData** function is used to create a **FormData** object from a ECMAScript object. The function takes a ECMAScript object as input and returns a **FormData** object.
