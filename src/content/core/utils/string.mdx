String utils are a set of utility functions that are designed to help with working with strings in ECMAScript.

## Capitalize

The **getCapitalizedString** function is a utility function that takes a string as input and returns a new string with the first letter capitalized. The function also has an optional second parameter **lowercase** which, if set to **true**, will convert the rest of the string to lowercase.

  console={100}
/>

## Camel Case

The **getCamelCaseString** function is a utility function that takes a string as input and returns a new string in camelCase format. The function iterates over the characters of the input string and converts them to lowercase, except for the first character of each word, which is converted to uppercase. The function uses regular expressions to identify the boundaries between words.
