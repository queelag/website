This is a collection of functions for working with FCM crypto.

## Create ECDH

The `createFcmECDH` functions creates a new ECDH instance with the prime256v1 curve and generates a new key pair.

  dependencies={{ '@aracna/core': 'latest', '@aracna/fcm': 'latest' }}
  files={SB_CREATE_FCM_ECDH}
  template='node'

  disableStackBlitz
/>
