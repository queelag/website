This is a collection of functions for working with image elements.

## Cache the data of an element

The `cacheImageElement` function caches the base64 data of an image element.

- Optionally the caching can be forced even if the image has already been cached.
- Optionally the quality and type of the saved data can be specified.

<CodeBlock
  client:load
  dependencies={{ '@aracna/core': 'latest', '@aracna/web': 'latest' }}
  files={SB_CACHE_IMAGE_ELEMENT}
  template='javascript'
  title='cacheImageElement'
  console={100}
/>
