This is a collection of functions for working with slider elements.

## Calculate a thumb element's percentage

The `getSliderThumbElementPercentage` function returns the percentage of a value between a minimum and maximum.

- Optionally the percentage can be rounded.
- Optionally the number of decimals can be specified.
