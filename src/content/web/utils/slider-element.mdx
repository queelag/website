Slider element utils are a set of utility functions that are used to work with slider elements in the browser environment. They can help to simplify the process of positioning the slider thumb element and calculating the percentage of the slider that the thumb should be positioned at.

## Get Thumb Element Percentage

The **getSliderThumbElementPercentage** function is used to get the percentage of the slider that the thumb should be positioned at based on a given value, minimum value, maximum value, and number of decimals. This can be useful for positioning the slider thumb element dynamically based on user input or other factors.
