This is a collection of functions for working with radio elements.

## Find a button by value

The `findRadioButtonByValue` function finds a radio button by its value.
Optionally a custom function can be passed to get the value of the radio button.

## Find a button's label by value

The `findRadioButtonLabelByValue` function finds a radio button label by its value.
