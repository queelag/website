This is a collection of functions for working with select elements.

## Find an option by value

The `findSelectOptionByValue` function finds a select option by its value.
Optionally a custom function can be passed to get the value of the select option.

## Find an option's label by value

The `findSelectOptionLabelByValue` function finds a select option label by its value.

- Optionally a custom function can be passed to get the label of the select option.
- Optionally a custom function can be passed to get the value of the select option.
