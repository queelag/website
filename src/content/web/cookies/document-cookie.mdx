The `DocumentCookie` class is a `Cookie` that uses the `document.cookie` API.

## Set an item

The `set` method in the `Cookie` class sets an item in the cookies. Optionally, you can specify the keys of the item that you want to set. If you don't specify any key, the whole item will be set.

<CodeBlock
  client:load
  dependencies={{ '@aracna/core': 'latest', '@aracna/web': 'latest' }}
  files={SB_DOCUMENT_COOKIE_SET}
  template='javascript'
  title='DocumentCookie'
  console={100}
/>
