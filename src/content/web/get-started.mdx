The **@aracna/web** package depends on the **@aracna/core** package as a peer dependency, which is required for it to function properly.

## NPM

<CodeWindow client:load language='bash'>
  npm install --save @aracna/web
</CodeWindow>

## Yarn

<CodeWindow client:load language='bash'>
  yarn add @aracna/web
</CodeWindow>
