The **FormFieldElementCollector** class is a utility class that provides methods for storing and retrieving instances of **FormFieldElement** subclasses. It uses three static maps, **mapByID**, **mapByTarget**, and **mapByUID**, to store elements by their id, target and uid properties, respectively.
