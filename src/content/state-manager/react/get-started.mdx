The **@aracna/state-manager-react** package has dependencies on four other packages: **@aracna/core**, **@aracna/react**, **@aracna/state-manager** and **@aracna/web**. These packages are required for the **@aracna/state-manager-react** package to function properly.

## NPM

<CodeWindow client:load language='bash'>
  npm install --save @aracna/state-manager-react
</CodeWindow>

## Yarn

<CodeWindow client:load language='bash'>
  yarn add @aracna/state-manager-react
</CodeWindow>
