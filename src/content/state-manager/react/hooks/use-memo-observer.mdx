The `useMemoObserver` hook re-renders whenever any of the values it references from an observable inside the render function change.
Optionally the memoization dependencies can be specified.

<Alert client:load severity='info'>
  The main difference between **useObserver** and **useMemoObserver** is that the latter uses memoization to optimize performance.
</Alert>
