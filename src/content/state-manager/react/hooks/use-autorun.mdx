The `useAutorun` hook runs an effect immediately and re-runs it whenever any of the values it references from an observable change.
Optionally the dependencies can be specified.

  client:load
  dependencies={{
    '@aracna/core': 'latest',

    '@aracna/web': 'latest'
  }}
  files={SB_USE_AUTORUN}
