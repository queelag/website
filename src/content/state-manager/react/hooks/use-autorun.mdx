The **useAutorun** hook is a custom hook that is designed to be used with React functional components. It provides a way to automatically re-run a function whenever one of its dependencies changes.

The **useAutorun** hook takes a single argument, which is a function that will be executed whenever one of its dependencies changes. This function is passed to the **autorun** function from the **@aracna/state-manager** package, which is responsible for tracking the dependencies and re-running the function when necessary.
