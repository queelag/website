The **@aracna/react-components** package has dependencies on four other packages: **@aracna/core**, **@aracna/web**, **@aracna/web-components** and **@aracna/react**. These packages are required for the **@aracna/react-components** package to function.

<PackageManagerBlock
  client:load
  install={{ dependencies: ['@aracna/core', '@aracna/react', '@aracna/web', '@aracna/web-components'], package: '@aracna/react-components' }}
/>
